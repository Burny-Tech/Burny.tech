```sh


\r‘: command not found 解决办法

当我们执行 shell 脚本的时候提示’\r’: command not found，但是检查了很多次并没有发现什么问题。
原因是 windows 下的换行符是\r\n，而 Linux 下的换行符是\n，所以只要执行下面的命令把\r 去掉就可以了。
解决方法：

#转化为unix格式
sed -i 's/\r$//' <filename> 
```
