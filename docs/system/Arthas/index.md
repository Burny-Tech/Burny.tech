## 可解决问题

1. 类从哪个jar包中加载，为什么会报各种Exception
2. 改的代码有没有执行到？
3. 无需增加日志在重新发布生jar包到身缠环境
4. 可以线上debug
5. 是否有一个全局视角来查看系统的运行情况
6. 监控JVM实时运行的状态
7. 快速定位应用的热点，生成火焰图
